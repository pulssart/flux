# Flux

Smart RSS reader.

Développement local

```
npm install
npm run dev
```

Ouvre http://localhost:3000

Déploiement Netlify

- Node 20 recommandé (`.nvmrc` fourni)
- `netlify.toml` configuré avec `@netlify/plugin-nextjs`
- Si erreur de cache Netlify: dans l’interface Netlify, Clear cache and deploy site
